# PySphere Assignment 1: Ohm’s Law

## 📌 Description
This project is a simple Python script to calculate **Voltage (V)** using **Ohm's Law**:

\[
V = I \times R
\]

Where:
- `V` = Voltage in volts
- `I` = Current in amperes
- `R` = Resistance in ohms
